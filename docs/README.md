# Copha
## Copha is a general framework for running automated network tasks.

> ⚠️ [Usage Warning](https://copha.net/usage_warning)

# Getting Started
## Installation
```
npm i -g copha
# or "yarn global add copha"
```

## How to use
```
copha -h
```
more help about command line [here](https://copha.net/cli)
