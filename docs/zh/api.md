	* [获取项目列表](#获取项目列表)

	* 获取存储模块列表

	* 关闭API服务

## API 具体说明
### 获取系统配置
`GET` `/api/settings`​

### 获取项目列表
`GET` `/api/projects`

### 获取项目配置
`GET` `/api/projects/{name}`
